# genesis

The application allows you to go through authorization, get an access token, which is used to create deal, contact or company entities using the amoCRM API.

Stack:
<ol>
<li>Vue3</li>
<li>Vue-router</li>
<li>Vuex</li>
<li>Axios</li>
</ol>

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
